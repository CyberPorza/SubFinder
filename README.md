# 🛡️ CyberPorza-SubFinder v1.1

A fast and efficient Subdomain Discovery tool developed with Python. Designed for security researchers and ethical hackers to identify hidden subdomains and expand the attack surface of target systems.

## ✨ Features
- **Rapid Discovery:** Checks common subdomains in seconds with optimized request timeouts.
- **Visual Interface:** Professional terminal experience with CyberPorza ASCII Art.
- **Status Tracking:** Automatically reports HTTP response codes for discovered subdomains.
- **Error Handling:** Robust architecture resilient to connection errors and user interrupts.

## 🚀 Installation and Usage

1. Clone the repository directly from the terminal:
   
   ```bash
   git clone [https://github.com/CyberPorza/CyberPorza-SubFinder.git](https://github.com/CyberPorza/CyberPorza-SubFinder.git)

3. Navigate to the project directory:

   ```bash
   cd CyberPorza-SubFinder

5. Run the tool with the target domain:

   ```bash
   python3 finder.py example.com
